# Virtual Companion for Behavioural Health


https://www.hackerearth.com/challenges/hackathon/ust-d3code-campus-hackathon23/


A virtual companion for behavioural health. Provides immediate support and assitance, which helps counter fear of judgement, social stigma, etc. Makes use of CBT (cognitive behavioural therapy) dataset and face expression models to derive the mental state. Displays a 3D human-like blender avatar for interacting with the user.
